Personal website

Mijn eigen website is de eerste website die ik helemaal zelf heb ontworpen, getest en heb gerealiseerd. Ik heb deze site gemaakt zodat mensen op een makkelijke manier informatie over mij kunnen vinden. In de website presenteer ik mezelf, vertel ik wat ik doe en laat ik de projecten zien waar ik het meest trots op ben.

Ik heb deze website als project voor mijn studie kunnen maken. Hiervoor heb ik onderzoek gedaan naar welke informatie in de website moet en hoe die informatie gepresenteerd moet worden. Vervolgens heb ik een ontwerp en prototype gemaakt in Adobe XD. Het prototype is getest door een aantal gebruikers en zo heb ik er nog een aantal interactiefoutjes uit kunnen halen. Tot slot heb ik de site gerealiseerd. Hiervoor heb ik alleen nog maar HTML, CSS en Javascript gebruikt. In de toekomst wil ik frameworks als Vue.js en Wordpress onder de knie krijgen. Zodra ik deze technieken beheers, pas ik ze toe in mijn website. Zo representeert de website altijd mijn skills op het gebied van webdevelopment.
